How it works

Every image gets a random, globally unique 10-digit name (extension kept). Files already named that way are skipped, as are files renamed in a previous run. Non-images are ignored. A history log keeps every old → new mapping.

Rename before tagging — renaming afterwards orphans the tag JSONs.
